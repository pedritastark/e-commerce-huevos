# Single Product Page - Aura

Proyecto en blanco optimizado para crear una Single Product Page de alto rendimiento y SEO-friendly.

## Stack Tecnológico

- **Framework**: Vite + React + TypeScript
- **Estilos**: Tailwind CSS v4
- **Iconos**: Lucide React
- **Animaciones**: Framer Motion

## Características

- Hot Module Replacement (HMR) instantáneo
- TypeScript para seguridad de tipos
- Tailwind CSS para diseño responsivo rápido
- Iconos modernos y ligeros con Lucide
- Animaciones fluidas con Framer Motion
- Optimizado para SEO y rendimiento

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor de desarrollo estará disponible en `http://localhost:5173`

## Build para Producción

```bash
npm run build
```

## Preview de Producción

```bash
npm run preview
```

## Estructura del Proyecto

```
src/
  ├── App.tsx          # Componente principal de la página de producto
  ├── index.css        # Estilos globales con Tailwind
  └── main.tsx         # Punto de entrada de la aplicación
```

## Personalización

El componente `App.tsx` incluye una plantilla básica de Single Product Page con:

- Sección de imagen del producto
- Información del producto (nombre, descripción, precio)
- Sistema de calificación con estrellas
- Lista de características
- Botones de acción (Añadir al carrito, Favoritos)
- Animaciones de entrada suaves

Personaliza el contenido, colores y diseño según tus necesidades.
